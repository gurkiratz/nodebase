# Extra Features

## Editor

- Pressing `Cmd/Ctrl + S` will trigger a workflow save
- Workflows are also auto-saved after a short period of user inactivity ("debounced save")
- If you attempt to close or refresh the tab while there are unsaved changes in your workflow, an alert will appear to prevent accidental loss of work.
- Ctrl/Cmd + Z for undo
- Duplicate node - also with (option + drag)
  - copies exact configuration
  - handles for any errors such as previous node data
- Delete Node by pressing delete key after selecting it
